Design Patterns

  • Creational Patterns

    Instead of creating objects directly, these patterns give you more flexibility in how objects come into existence.

    Example:  Singleton, Builder, Factory

  • Structural Patterns

    Deals with how objects relate to each other.
    Think of them as blueprints for building large structures from individual pieces.

    Example:  Facade, Adapter

  • Behavioral Patterns

    Handles communication between objects - how they interact and distribute responsibility.

    Example:  Strategy, Observer
