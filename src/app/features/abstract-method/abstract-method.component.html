Abstract Method

In TypeScript, abstract methods are methods defined in an abstract class without implementation.  The concrete subclasses are then required to provide the actual implementation of these methods.  This is useful when you want to define a common interface or contract for a set of related classes while allowing each class to provide its own specific behavior.
